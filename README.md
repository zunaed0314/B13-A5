1. What is the difference between var, let, and const?

Ans: var: Function-scoped, can be redeclared and reassigned. Older way of declaring variables.
     let: Block-scoped, cannot be redeclared in the same scope but can be reassigned.
     const: Block-scoped, cannot be redeclared or reassigned. VAlue stays constant

2. What is the spread operator (...)?

Ans: The spread operator (...) is used to expand or spread the values of an array or object. Array is one object. But we may need each elements. So we use spread operator.

3. What is the difference between map(), filter(), and forEach()?

Ans: map(): Used when you want to change every item in an array and get a new array with the changed values.
     filter(): Used when you want to pick only some items from an array based on a condition.
     forEach(): Used when you just want to go through each item in the array and do something, but you don’t get a new array back.

4. What is an arrow function?

Ans: An arrow function is a shorter and cleaner way to write a function in JavaScript. If its done in one line, it doesn't require a separate return statement but if it is multi line, it requires a return statement. 

5. What is template literal?

Ans: Template literals are a new and easier way to write strings in JavaScript.
     Instead of using single ' ' or double " " quotes, we use this ` `. This lets you insert variables directly inside the string, which makes the code cleaner.
